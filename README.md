# Description
This is a repository for CSCE 548 Spring 2026. It is the repo for 
The semester-long project I will be tasked with creating and examining to see if AI can create secure software. It is using MySQL for database storage and is coded in Java (Springboot w/ Maven) for the backend

# Project 1 Instructions
To run the code, open a terminal in CSCE548/clientinfomanager and type, "mvn -q exec:java"
